split(splitter: str | List[str] | Tuple[str, ...] | None = None, overwrite: bool = False, cont_dim: dict | None = None, **inputs)ÔÉÅ

Run this task parametrically over lists of split inputs.

Parameters:

  • splitter (str or list[str] or tuple[str] or None) ‚Äì the fields which to split over. If splitting over multiple fields, lists of fields are interpreted as outer-products and tuples inner-products. If None, then the fields to split are taken from the keyword-arg names.

  • overwrite (bool, optional) ‚Äì whether to overwrite an existing split on the node, by default False

  • cont_dim (dict, optional) ‚Äì Container dimensions for specific inputs, used in the splitter. If input name is not in cont_dim, it is assumed that the input values has a container dimension of 1, so only the most outer dim will be used for splitting.

  • **split_inputs ‚Äì fields to split over, will automatically be wrapped in a StateArray object and passed to the node inputs

self ‚Äì a reference to the task
